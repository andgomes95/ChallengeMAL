Anime Pyramid Challenge

Venha participar desse desafio!

O Anime Pyramid Challenge é um desafio onde o objetivo é cumprir uma piramide que é dividida em 5 camadas, sendo que cada camada possui uma tarefa diferente, com uma quantidade de animes diferentes. A cada etapa, a quantia de animações passa a ser menor, porém, as missões se tornam mais desafiantes! Venha escalar está piramide com a gente!
